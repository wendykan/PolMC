CheckIfValidXForFFT 

bool CheckIfValidXForFFT(
    const std::valarray<Double>& a,
    long l,
    char *s);
Parameters
a
An array of real number
l
Line number from which function was called
s
Filename from which function was called
Return Value

Non zero if invalid.

Discussion

This function should not be called directly. See the macro CheckIfValidXForFFT_(). Check to see if a array of real numbers is appropriate for use with the FFT algorithm, that is, if the first element is zero, and if these elements are symmetric with respect to zero (i.e. a[i] == -a[a.size() - i]).
