PhotonBartel

Superclass: Photon


The class PhotonBartel implements the various actions done on a photon wavepacket according to

"Monte Carlo simulations of the diffuse backscattering Mueller matrix for highly scattering media", Bartel & Hielscher Appl. Opt. Vol 39, April 1st 2000, p. 1580.

The notation used in the implementation of this class is similar to the one used in the article to avoid confusion. The goal of this implementation is to be readable, not necessarily fast.
